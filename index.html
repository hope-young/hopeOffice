hope-Office

AI assistant for Word, Excel, PowerPoint MIT

This page is the GitHub Pages landing for the hope-Office Web Add-in. The actual product runs inside Word, Excel, or PowerPoint as a Custom Tab on the ribbon — it is not a web app you can use from a browser.

Sideload the add-in

Download manifest.xml (or use the URL above directly), then in any Office app: 

Insert → My Add-ins → Upload My Add-in → pick manifest.xml

The Custom Tab hope-Office appears in the ribbon with three groups: Chat, History, and Settings. Open Settings on first run to add your LLM API key.

How it's served

The task pane and all assets in this branch are statically served by GitHub Pages. No backend, no dev server, no local install of Node. The add-in pulls manifest.xml from this URL and Office loads the task pane from ./src/taskpane/index.html.

Compatibility

Web Add-in runs on Word, Excel, PowerPoint on Windows, macOS, and Office on the web. Not a Windows-only COM add-in, not a VSTO add-in — the Web Add-in model is the only one that crosses all three platforms.
